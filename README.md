http 1.1 server

this repository contains an rfc compliant http 1.1 server in java.
RFCs: https://datatracker.ietf.org/doc/html/rfc9112, https://datatracker.ietf.org/doc/html/rfc9110

The server is entirely built on top of tcp and doesn't use any http libarry / framework.
(only uses Sockets for communication)
